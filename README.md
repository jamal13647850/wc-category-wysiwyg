# WC Category WYSIWYG

WordPress mu-plugin that adds WYSIWYG editor to WooCommerce product category descriptions.

## Features

- Adds rich text editor to product category descriptions
- Supports full WordPress editor functionality
- Compatible with WooCommerce
- SEO-friendly output
- **Full Persian (fa_IR) localization support**

## Installation

1. Copy `wc-category-wysiwyg` folder to `wp-content/mu-plugins/`
2. Activate through WordPress admin or automatically (mu-plugins)

## Requirements

- WordPress 5.0+
- WooCommerce 3.0+
- PHP 7.4+
