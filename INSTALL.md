# WC Category WYSIWYG

## Installation Instructions

### Method 1: Manual Installation

1. Download the plugin files
2. Upload the entire `wc-category-wysiwyg` folder to your WordPress `wp-content/mu-plugins/` directory
3. The plugin will be automatically activated (mu-plugins don't need activation)

### Method 2: Git Clone

```bash
cd wp-content/mu-plugins/
git clone https://github.com/jamal13647850/wc-category-wysiwyg.git
```

### Method 3: Composer

```bash
composer require jamal13647850/wc-category-wysiwyg
```

## Usage

1. Go to WordPress Admin → Products → Categories
2. Add or edit a category
3. Use the WYSIWYG editor to format your description
4. Save the category
